# Rx Radar

Smart Drug Availability & Shortage Alert System

## Problem
Medicine shortages create delays and confusion for patients and pharmacies.

## Solution
Rx Radar helps users search medicines, check pharmacy availability, and receive shortage alerts in real time.

## Features
- Drug Search
- Pharmacy Availability
- Shortage Alerts
- Waitlist Notifications
- Crowd Verification

## Tech Stack
- HTML
- CSS
- JavaScript
- PHP
- MySQL
