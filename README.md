# 💊 Rx Radar – Smart Drug Availability & Shortage Alert System

Rx Radar is a full-stack healthcare web application that helps users search medicines across nearby pharmacies and check real-time stock availability. The platform is designed to reduce the difficulty patients face during medicine shortages by providing pharmacy-wise medicine inventory and live availability tracking.

---

# 🚨 Problem Statement

Patients often struggle to find essential medicines during shortages and are forced to visit multiple pharmacies. Pharmacies also face repeated inquiries regarding medicine availability.

Rx Radar solves this problem by creating a centralized smart medicine availability platform that connects users with pharmacies and provides real-time stock information.

---

# ✅ Features

- 🔍 Real-time medicine search
- 🏥 Pharmacy-wise stock availability
- 📦 Live inventory tracking system
- 🔐 Secure user authentication
- 📍 Location-based pharmacy information
- ⚡ Dynamic frontend-backend integration
- 🗄️ MySQL database management
- 🔄 SQL JOIN-based inventory search

# 🚀 How to Run

### 1️⃣ Move Project to XAMPP htdocs

```bash
/opt/lampp/htdocs/
```

### 2️⃣ Start Apache & MySQL

Open XAMPP and start:
- Apache
- MySQL

### 3️⃣ Import Database

- Open phpMyAdmin
- Create database named:

```bash
rx_radar
```

- Import:

```bash
database/schema.sql
```

### 4️⃣ Run Project

Open browser:

```bash
http://localhost/Rx-Radar/frontend/index.html
```
