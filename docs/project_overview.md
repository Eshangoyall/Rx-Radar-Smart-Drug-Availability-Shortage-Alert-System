# Rx Radar

## Overview
Rx Radar is a Smart Drug Availability & Shortage Alert System designed to help users find medicines in nearby pharmacies in real time.

## Problem
Patients often struggle to locate essential medicines during shortage situations.

## Solution
Rx Radar connects patients and pharmacies through a centralized platform that shows medicine availability and shortage alerts.

## Main Features
- Real-time medicine search
- Pharmacy availability
- Waitlist notifications
- Crowd verification
- Shortage alerts
